simil.art

Content-based search application

Get Started

Search by Content

Our site offers an advanced search feature that enables you to easily find artworks based on their content. This feature is especially useful if you're looking for a specific piece of art or if you want to explore new ones!

Visualize Feature Maps

Browse through what the neural network model perceives, namely the feature maps at a specific layer. This translates to the application of specific filters at each layer on the input image.

Refine Similar Images by Color

Refine your search results based on color similarity. We calculate the correlation between the color histograms of the uploaded image and the similar images, allowing you to focus on images with similar color distributions.

Technologies Used

FAISS

We used FAISS for indexing the feature vectors of the images.

Neural Networks

We used ResNet with PyTorch for extracting features from the images.

RESTful API

We used Django to create the RESTful API and manage the image database.

Coming Soon

A sneak peek of upcoming features and updates.

Image Dataset Upload

We are considering the possibility of allowing users to upload their own image datasets directly to the site. This will enable users to work on their own datasets, perform searches, refine results, and more.

Refinement Features

We plan to introduce new refinement features to enhance search results. These features may include refinement by texture, shape, and the display of histograms and other information used by the algorithms.

Model Comparison

We will soon provide the ability to compare the performances and accuracies of multiple models. This will allow users to make informed decisions about which model best suits their needs.

CLIP Integration

We are planning to integrate CLIP for text-image search functionality. This will enable users to search for images using textual descriptions, making the search process more flexible.
